# TXM
TXM (Terminal TeX Math) is a math rendering engine with LaTeX support.

## Quick run

```
nix run github:thatmagicalcat/txm -- "E = mc^2"
```

Requires [Nix](https://nix.dev/install-nix) with flakes enabled.
